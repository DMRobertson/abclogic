How to play

Rules

ABC Logic is a single-player logic puzzle game. It takes place on a square grid, where some rows and columns are annotated with letters. The objective is to fill the grid with the letters A, B, C such that:

  • Each column contains each letter exactly once.
  • Each row contains each letter exactly once.
  • An annotated letter to the left of a row is the first letter in that row. Similarly, a letter to the right of a row is the last letter of that row.
  • An annotated letter above a column is the first letter in that column. Similarly, a letter below a column is the last letter of that row.

Notation

If we determine that a square contains a letter, we mark it with that letter. If we don't know what a square contains, we leave it blank. If we are certain that the square doesn't contain a letter, we mark it with a ✘.

If a mistake is made—i.e. if one of the rules above is broken—then the problem row or column will be highlighted in red.

Controls

Click or tap on a given cell to select it. Keyboard users can navigate using the ↑←↓→ keys.
